Політика конфіденційності

Останнє оновлення: березень 2026

У цьому документі пояснюється, які дані RustInfo збирає, навіщо це потрібно та як ви можете керувати своїми даними під час використання сайту, бота та повʼязаних сервісів.

1. Які дані ми збираємо

Через Discord OAuth:

  • Discord ID, імʼя профілю, аватар і список серверів, де у вас є права керування.

Під час підключення Rust+:

  • Steam ID для ідентифікації вашого акаунта в Rust+;
  • Rust+ JWT-токен (Facepunch) для реєстрації push-сповіщень;
  • FCM / Expo токени для доставки сповіщень про події.

Технічні дані:

  • IP-адреса та User-Agent для безпеки й діагностики;
  • знеособлена аналітика відвідувань сайту (Google Analytics).

2. Як ми використовуємо дані

  • Для роботи основних функцій RustInfo: моніторинг, сповіщення, керування пристроями;
  • для привʼязки Discord-акаунта до серверів і Steam ID;
  • для захисту від зловживань та порушення правил;
  • для покращення якості сервісу та інтерфейсу.

3. Передача третім сторонам

Ми не продаємо персональні дані. Передача відбувається лише в межах необхідних інтеграцій:

  • Discord — для авторизації та базових даних профілю;
  • Facepunch / Rust+ API — для роботи інтеграції Rust+;
  • Firebase Cloud Messaging / Expo — для push-сповіщень;
  • Google Analytics — для агрегованої веб-аналітики.

4. Термін зберігання

Дані зберігаються лише стільки, скільки потрібно для роботи сервісу. Ви можете подати запит на видалення через наш Discord-сервер підтримки.

5. Ваші права

  • Відкликати доступ у налаштуваннях Discord;
  • припинити користування сервісом у будь-який момент;
  • запросити видалення ваших збережених даних.

6. Cookies і localStorage

Ми використовуємо cookies і localStorage для сесій входу та збереження налаштувань (наприклад, теми).

7. Безпека

Ми застосовуємо стандартні заходи захисту, зокрема HTTPS та контроль доступу. Абсолютної безпеки в інтернеті не існує.

8. Оновлення політики

Ми можемо оновлювати політику в міру розвитку сервісу. Актуальна версія завжди доступна на цій сторінці.

9. Контакти

З питань конфіденційності звертайтеся в наш Discord-сервер підтримки.
