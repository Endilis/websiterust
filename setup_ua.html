Підключення бота

Слідуйте цій інструкції для підключення бота RustInfo

🚀 Початок роботи

Для повноцінного використання функцій Rust+ вам знадобиться браузер Google Chrome та кілька простих кроків з налаштування.

1

Додати бота в свій Discord канал

Спершу необхідно додати RustInfo бота на ваш Discord сервер.

Додати бота на сервер →
1.2

Почати використання бота

Напишіть команду !setup в будь-якому каналі вашого Discord сервера, щоб почати користуватися ботом.

2

Встановлення браузера Chrome

Вам знадобиться браузер Google Chrome для роботи з розширенням Rust+.

2.1

Встановлення розширення для браузера

Встановіть офіційне розширення Rust+py Link Companion з магазину Chrome.

Встановити розширення →
2.2

Авторизація на сайті Facepunch

Після встановлення розширення необхідно авторизуватись на офіційному сайті Facepunch Companion.

Перейти на сайт Facepunch →
2.3

Копіювання даних для підключення

Після авторизації на сайті Facepunch натисніть кнопку Copy information для копіювання даних.

2.4

Підключення до бота

Натисніть на кнопку Rust+ в меню бота і вставте скопійовану інформацію з попереднього пункту.

💡 Важливо:

Знайдіть кнопку Rust+ в меню бота і вставте скопійовані дані для завершення підключення.

2.5

Прив'язка в грі

Зайдіть на сервер. Натисніть Esc у верхній частині екрану, виберіть Поточний сервер і натисніть Прив'язати.

2.6

Підтвердження підключення

Після успішного підключення ви побачите повідомлення від бота. Протягом хвилини ви зможете почати користуватись ботом.

✅ Готово!

Тепер ви можете використовувати всі функції Rust+ через Discord бота. Для отримання списку команд використовуйте .help
