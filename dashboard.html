Тот же формат, что расширение RustInfo шлёт на сервер в POST /auth/fcm — привязка Steam/Rust+ и FCM для выбранного сервера Discord. Установите Rustplus.py Link Companion, скопируйте JSON из расширения и вставьте ниже. Стороннее расширение со страницы не детектируется — блок доступен всегда.

Как привязать Rust+ токен
  1. 1
    Установите расширение RustInfo Bot в Chrome (Developer mode → Load unpacked)
  2. 2
    Иконка расширения → companion-rust.facepunch.com/login
  3. 3
    Войдите через Steam
  4. 4
    Токен отправится на сервер — вас вернёт в панель

    Сохраняется по вашему Discord. Отключает сообщения бота в игре для типов ниже. Работает, если Steam в панели совпадает с привязкой Rust+ к серверу.

    Сообщения в каналах Discord (алерты, события) настраиваются на самом сервере: кнопки под сообщением бота и команды в каналах. Внутриигровые тумблеры слева относятся только к чату Rust+ в игре.
