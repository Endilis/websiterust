Нажмите кнопку "Добавить бота" на этой странице. Выберите ваш Discord сервер из списка и предоставьте необходимые разрешения. RustInfo автоматически создаст нужные каналы.

Откройте приложение Rust+ на телефоне. Перейдите в настройки сервера и получите pairing code. Введите код в Discord командой !pair [код]. Готово!

Часто Задаваемые Вопросы о RustInfo

❓ Как добавить RustInfo бота на мой Discord сервер?

Нажмите кнопку "Добавить бота" в верхней части страницы или по этой ссылке. Выберите ваш Discord сервер и предоставьте нужные разрешения. Бот автоматически создаст каналы и будет готов к работе.

❓ RustInfo действительно бесплатный?

Да! RustInfo полностью бесплатный Discord бот для мониторинга Rust+ серверов. Все функции доступны без ограничений: Smart Switch управление, статус игроков, уведомления о рейдах, мониторинг 24/7. Никаких платных тарифов или подписок!

❓ Какие команды есть у RustInfo?

RustInfo поддерживает широкий набор команд: !status (статус сервера), !players (список игроков), !switch (управление Smart Switch), !map (карта сервера), !pair (подключение к Rust+) и многое другое. Полный список команд доступен в документации.

❓ Как управлять Smart Switch через RustInfo?

После подключения к Rust+ серверу используйте команду !switch чтобы увидеть список всех Smart Switch устройств. Кликните на кнопки в Discord сообщении чтобы включить или выключить любое устройство. Это работает моментально!

❓ На скольких Rust+ серверах можно использовать бота?

RustInfo поддерживает мониторинг нескольких Rust+ серверов одновременно. Вы можете подключить бота к неограниченному количеству серверов и управлять всеми из одного Discord канала.

❓ Безопасно ли использовать RustInfo?

Абсолютно! RustInfo использует официальный Rust+ Companion API от Facepunch. Бот не запрашивает пароли Steam, не имеет доступа к вашему аккаунту и работает только через публичное API. Это полностью безопасно.
