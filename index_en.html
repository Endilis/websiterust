How to Start Using RustInfo Discord Bot for Rust+?

Setting up RustInfo takes just a few minutes. Follow the simple step-by-step instructions:

1️⃣ Add Bot to Discord

Click the "Add Bot" button on this page. Select your Discord server from the list and grant the necessary permissions. RustInfo will automatically create the needed channels.

2️⃣ Connect Rust+ Server

Open the Rust+ app on your phone. Go to server settings and get the pairing code. Enter the code in Discord with command !pair [code]. Done!

3️⃣ Configure Notifications

Choose which events you want to track: raids, online players, Smart Switch status. Set up channels for different notification types. Use the !settings command.

4️⃣ Enjoy Automation!

Now RustInfo automatically monitors your Rust+ server 24/7. Get notifications, manage devices, track players. Full control from Discord!

Frequently Asked Questions about RustInfo

❓ How to add RustInfo bot to my Discord server?

Click the "Add Bot" button at the top of the page or this link. Select your Discord server and grant the required permissions. The bot will automatically create channels and be ready to work.

❓ What commands does RustInfo have?

RustInfo supports a wide range of commands: !status (server status), !players (player list), !switch (Smart Switch control), !map (server map), !pair (connect to Rust+) and much more. Full command list is available in the documentation.

❓ How to control Smart Switch through RustInfo?

After connecting to a Rust+ server, use the !switch command to see a list of all Smart Switch devices. Click buttons in the Discord message to turn any device on or off. It works instantly!

❓ How many Rust+ servers can I monitor with the bot?

RustInfo supports monitoring multiple Rust+ servers simultaneously. You can connect the bot to unlimited servers and manage all from one Discord channel.

❓ Is it safe to use RustInfo?

Absolutely! RustInfo uses the official Rust+ Companion API from Facepunch. The bot doesn't ask for Steam passwords, has no access to your account and works only through the public API. It's completely safe.

❓ Where to get setup help?

Join our Discord support server! There you'll find detailed guides, video tutorials and can ask questions to experienced users. Also check the setup page with step-by-step instructions.
