Политика конфиденциальности

Последнее обновление: март 2026

В этом документе мы простыми словами объясняем, какие данные получает RustInfo, зачем они нужны и как вы можете управлять ими. Используя сайт, бота и связанные сервисы RustInfo, вы принимаете эту Политику.

1. Какие данные мы собираем

Через Discord OAuth:

  • Discord ID, имя профиля, аватар и список серверов, где у вас есть права управления.

При подключении Rust+:

  • Steam ID для связи аккаунта с Rust+ API;
  • JWT-токен Rust+ (Facepunch) для регистрации push-уведомлений;
  • FCM / Expo токены для доставки уведомлений о событиях.

Техническая информация:

  • IP-адрес и User-Agent для безопасности и диагностики;
  • обезличенная статистика посещений (Google Analytics).

2. Как мы используем данные

  • Чтобы предоставлять функции RustInfo: мониторинг, уведомления, управление устройствами;
  • для привязки Discord-аккаунта к серверу и Steam ID;
  • для защиты от злоупотреблений и нарушения правил;
  • для улучшения качества сервиса и интерфейса.

3. Передача данных третьим сторонам

Мы не продаём персональные данные. Передача ограничена необходимыми интеграциями:

  • Discord — для авторизации и базовой информации профиля;
  • Facepunch / Rust+ API — для работы интеграции Rust+;
  • Firebase Cloud Messaging / Expo — для push-уведомлений;
  • Google Analytics — для агрегированной веб-аналитики.

4. Срок хранения

Данные хранятся только столько, сколько нужно для работы сервиса. Вы можете запросить удаление привязок и технических данных через наш Discord-сервер поддержки.

5. Ваши права

  • Отозвать доступ приложения в настройках Discord;
  • прекратить использование сервиса в любой момент;
  • обратиться с запросом на удаление данных.

6. Cookies и localStorage

Мы используем cookies и localStorage для авторизации, сессии и пользовательских настроек (например, темы интерфейса).

7. Безопасность

Мы применяем стандартные меры защиты, включая HTTPS и ограничение доступа к данным. Однако абсолютной безопасности в интернете не существует.

8. Изменения политики

Мы можем обновлять документ. Актуальная версия всегда опубликована на этой странице с датой последнего изменения.

9. Контакты

Если у вас есть вопросы по обработке данных, напишите нам в Discord-сервер поддержки.
