Подключение бота

Следуйте этой инструкции для подключения RustInfo бота

🚀 Начало работы

Для полноценного использования функций Rust+ вам потребуется браузер Google Chrome и несколько простых шагов по настройке.

1

Добавить бота в свой Discord канал

Первым делом необходимо добавить RustInfo бота на ваш Discord сервер.

Добавить бота на сервер →
1.2

Начать использование бота

Напишите команду !setup в любом канале вашего Discord сервера, чтобы начать пользоваться ботом.

2

Установка браузера Chrome

Вам понадобится браузер Google Chrome для работы с расширением Rust+.

2.1

Установка расширения для браузера

Установите официальное расширение Rust+py Link Companion из магазина Chrome.

Установить расширение →
2.2

Авторизация на сайте Facepunch

После установки расширения необходимо авторизоваться на официальном сайте Facepunch Companion.

Перейти на сайт Facepunch →
2.3

Копирование данных для подключения

После авторизации на сайте Facepunch нажмите кнопку Copy information для копирования данных.

2.4

Подключение к боту

Нажмите на кнопку Rust+ в меню бота и вставьте скопированную информацию из предыдущего пункта.

💡 Важно:

Найдите кнопку Rust+ в меню бота и вставьте скопированные данные для завершения подключения.

2.5

Привязка в игре

Зайдите на сервер. Нажмите Esc в верхней части экрана, выберите Текущий сервер и нажмите Привязать.

2.6

Подтверждение подключения

После успешного подключения вы увидите сообщение от бота. В течение минуты вы сможете начать пользоваться ботом.

✅ Готово!

Теперь вы можете использовать все функции Rust+ через Discord бота. Для получения списка команд используйте .help
