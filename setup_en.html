Follow these instructions to connect the RustInfo bot

🚀 Getting Started

To fully use Rust+ features, you'll need Google Chrome browser and a few simple setup steps.

1

Add Bot to Your Discord Server

First, you need to add the RustInfo bot to your Discord server.

Add Bot to Server →
1.2

Start Using the Bot

Type the !setup command in any channel of your Discord server to start using the bot.

2

Install Chrome Browser

You'll need Google Chrome browser to work with the Rust+ extension.

2.1

Install Browser Extension

Install the official Rust+py Link Companion extension from the Chrome Web Store.

Install Extension →
2.2

Authorize on Facepunch Website

After installing the extension, you need to authorize on the official Facepunch Companion website.

Go to Facepunch Website →
2.3

Copy Connection Data

After authorization on the Facepunch website, click the Copy information button to copy the data.

2.4

Connect to Bot

Click the Rust+ button in the bot menu and paste the copied information from the previous step.

💡 Important:

Find the Rust+ button in the bot menu and paste the copied data to complete the connection.

2.5

In-Game Pairing

Join the server. Press Esc at the top of the screen, select Current Server and click Pair.

2.6

Connection Confirmation

After successful connection, you'll see a message from the bot. Within a minute, you'll be able to start using the bot.

✅ Done!

Now you can use all Rust+ features through the Discord bot. For a list of commands, use .help
