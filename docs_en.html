RustInfo Documentation

Complete guide to using the bot and its features

Welcome to the 𝐑𝐮𝐬𝐭𝐈𝐧𝐟𝐨 bot support community

RustInfo is a multifunctional Discord bot for monitoring Rust servers, tracking players, and managing devices via Rust+.

📈 Server Monitoring

Displays information about the selected server and updates data every 10-30 seconds:

  • Server status
  • Online player count
  • Server load

🎮 Player Search

Quick player search by name on the selected server.

🟢 Player Status

Displays player list and their online/offline status updated every 2-4 minutes.

Notifications:

  • + Enable notifications for tracked players
  • - Disable notifications for tracked players

Rust+ Commands

📌 Important: Rust+ is only available for users with the @Tester role. Type !tester in the general chat to automatically receive the role.
.help

Displays the list of available commands

Commands: .help, .add <name>, .remove <name>, .devices, .sw <name> <0|1>, .pop, .time

.pop

Shows current server population, including queue and slot count

.time

Shows current server time and remaining minutes until sunrise or sunset

⚠️ Before using the .add command, you must pair the device through the Rust+ app!
.add <device name>

Add a device to the bot's database. After executing the command, you'll receive a message confirming successful device addition.

.devices

Display list of devices you've connected. Output is displayed with a 1-second delay.

.remove <device name> or .remove all

Remove unused devices. Use .remove all to remove all devices.

.sw <device name> <0|1>

Turn on <1> or off <0> a smart switch
