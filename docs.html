Документация RustInfo

Полное руководство по использованию бота и его возможностей

Приветствуем в сообществе поддержки бот - 𝐑𝐮𝐬𝐭𝐈𝐧𝐟𝐨

RustInfo — многофункциональный Discord бот для мониторинга Rust серверов, отслеживания игроков и управления устройствами через Rust+.

📈 Мониторинг сервера

Показывает информацию о выбранном сервере и обновляет данные каждые 10-30 секунд:

  • Статус сервера
  • Количество онлайн игроков
  • Загруженность сервера

🎮 Поиск игроков

Быстрый поиск игроков по имени на выбранном сервере.

🟢 Статус игроков

Отображает список игроков и их статус онлайн/оффлайн с обновлением каждые 2-4 минуты.

Уведомления:

  • + Включайте уведомления об отслеживаемых игроках
  • - Выключайте уведомления об отслеживаемых игроках

Rust+ Команды

📌 Важно: Rust+ доступен только для пользователей с ролью @Тестер. Напишите !tester в общем чате для автоматической выдачи роли.
.help

Выводит список доступных команд

Команды: .help, .add <name>, .remove <name>, .devices, .sw <name> <0|1>, .pop, .time

.pop

Показывает текущий онлайн на сервере, включая очередь и количество слотов

.time

Показывает текущее время на сервере и оставшееся время в минутах до рассвета или заката

⚠️ Прежде чем использовать команду .add, вы должны связать устройство через приложение Rust+!
.add <имя устройства>

Добавление устройства в базу данных бота. После выполнения команды вы получите сообщение об успешном добавлении устройства.

.devices

Вывод списка устройств, которые вы подключали. Вывод производится с задержкой в 1 секунду.

.remove <имя устройства> или .remove all

Удаление неиспользуемых устройств. Используйте .remove all чтобы удалить все устройства.

.sw <имя устройства> <0|1>

Включение <1> или Выключение <0> умного выключателя
