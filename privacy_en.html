Privacy Policy

Last updated: March 2026

This policy explains what data RustInfo collects, why we collect it, and how you can manage your data when using our website, Discord bot, and related services.

1. Data we collect

Via Discord OAuth:

  • Discord ID, username, avatar, and server list where you have management permissions.

When connecting Rust+:

  • Steam ID to identify your Rust+ account;
  • Rust+ JWT token (Facepunch) for push registration and API connection;
  • FCM / Expo push tokens for notifications.

Technical data:

  • IP address and User-Agent for security and diagnostics;
  • anonymous traffic analytics (Google Analytics).

2. How we use data

  • To provide core RustInfo features (monitoring, alerts, device control);
  • to link your Discord account to server settings and Steam ID;
  • to prevent abuse and enforce platform rules;
  • to improve service quality and UX.

3. Third-party sharing

We do not sell personal data. Data is shared only where required for functionality:

  • Discord for authentication and profile basics;
  • Facepunch / Rust+ API for Rust+ integration;
  • Firebase Cloud Messaging / Expo for push delivery;
  • Google Analytics for aggregate website analytics.

4. Data retention

Data is kept only as long as necessary to operate the service. You may request deletion via our Discord support server.

5. Your rights

  • Revoke app access in Discord settings;
  • stop using the service at any time;
  • request deletion of your stored data.

6. Cookies and localStorage

We use cookies and localStorage for login sessions and preferences (for example, theme settings).

7. Security

We apply standard safeguards such as HTTPS and restricted access controls. No online system can be guaranteed 100% secure.

8. Policy updates

We may update this policy as the service evolves. The latest version is always available on this page with an updated date.

9. Contact

For privacy questions, contact us through our Discord support server.
